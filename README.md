📊 Financial Analysis Project

Projeto de análise financeira desenvolvido em Python para processamento e análise de dados financeiros armazenados em arquivos CSV.

O sistema realiza cálculos automáticos sobre lucros e perdas mensais, fornecendo métricas importantes para análise financeira.

📌 Prévia do Projeto

O programa realiza automaticamente:

✔ Total de meses analisados
✔ Total líquido de lucros/prejuízos
✔ Média dos lucros/perdas
✔ Média das mudanças mensais
✔ Maior aumento nos lucros
✔ Maior redução nos lucros

🚀 Exemplo de Saída
Análise Financeira
------------------------------
Total de Meses: 86
Total Líquido: R$38382578
Média dos Lucros/Perdas: R$446309.05
Média das Mudanças: R$-2315.12
Maior Aumento nos Lucros: Feb-2012 (R$1926159)
Maior Redução nos Lucros: Sep-2013 (R$-2196167)

📋 Download e Instalação

Para começar a utilizar o projeto, escolha uma das opções abaixo:

Clonar o repositório
git clone git@github.com:Deangellisberg/Projeto-financeiro.git

🔧 Instalação
Acesse a pasta do projeto
cd Projeto-financeiro
Execute o projeto
python financeiro.py
